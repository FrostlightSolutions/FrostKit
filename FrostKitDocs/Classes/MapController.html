MapController

The map controller handles basic map options and controls for a MKMapView. It provides automatic functions for adding/removing annotations, finding directions, zooming the map view and searching the annotations plotted on the map.This class is designed to be subclassed if more specific actions, such a refining the standard search or customising the annotations plotted.
